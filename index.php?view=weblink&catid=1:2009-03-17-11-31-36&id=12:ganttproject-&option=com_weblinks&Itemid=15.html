GanttProject 2.6.6

Just run the executable. It will install GanttProject and associate it with .gan files and optionally with Microsoft Project files

Java Runtime version 1.6 or higher is required for running GanttProject. You can get it from Oracle

Some anti-virus software may report a virus in the downloaded file. It is false-positive. We don't put any viruses into GanttProject. We recommended that you update your anti-virus signature file or contact your anti-virus client software and advise them of the false-positive statement from your anti-virus client software. If you're still concerned, consider checking file integrity with MD5 verifier (there are many on the Internet) or downloading a generic platform-independent ZIP file, which has no Windows executable files.

GanttProject 2.6.6

Open dmg file, and either drag GanttProject to your Applications or start it directly from the volume. It will register an icon for .gan files and associate them with GanttProject.

Java Runtime version 7 from Oracle is recommended for running GanttProject, although GanttProject can run with the older Java 6 from Apple

GanttProject 2.6.6

Debian package which we compile for the latest Ubuntu Linux, but which hopefully will work on other Debian-based distros. Download it and install as you would install any other .deb package.

If you're running some other Linux distribution, you may try platform-independent ZIP archive with a launcher written in bash

Java Runtime version 1.6 or higher is required for running GanttProject. You can get either Java Runtime from Oracle, or install OpenJDK from Ubuntu software center. sudo apt-get install openjdk-6-jre will also do the trick.

GanttProject 2.6.6

Make sure that Java Runtime Environment is installed and that you can run java from the command line. Download ZIP file, unzip it in your home directory ($HOME) and run a launcher script (one of ganttproject.sh, ganttproject.bat, ganttproject.command, depending on your platform) from $HOME/ganttproject-2.6.6 directory.

Read more about ZIP package on our Wiki.

Try GanttProject without installing

You need a web browser which can run WebStartable applications. Usually Web Start support appears in the most popular browsers automatically when you install the latest Java Runtime.

If WebStart is installed, just click the button above to start GanttProject demo. Please note that you may be asked to trust our security certificate and grant all access privileges.

For download time reasons, GanttProject WebStart edition which is hosted on our site has no HTML/PDF export and Microsoft Project™ import/export functions. As for the rest, it is a full featured GanttProject.
